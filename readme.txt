== Description ==

This plugin will show information about pray (shollu) times in your place. Times is given by calculating Longitude, Latitude of your location.

It’s may not suitable for extreme place like polar which has noon and night duration for months. So contact nearby mosque

== Installation ==

1. Upload the whole plugin folder to your /wp-content/plugins/ folder.

2. Go to the Plugins page and activate the plugin.

3. Go to Setting >> Shollu settings. Change your options.

4. Goto Widget page. Add the widget to your sidebar.
